    <p>&lt;h1 id=“label-doiref”&gt;doiref&lt;span&gt;&lt;a href=“#label-doiref”&gt;&amp;para;&lt;/a&gt; &lt;a href=“#documentation”&gt;&amp;uarr;&lt;/a&gt;&lt;/span&gt;&lt;/h1&gt;</p>

    <p>&lt;p&gt;&lt;strong&gt;this is alpha software, so expect changes&lt;/strong&gt;&lt;/p&gt;</p>

    <p>&lt;h3 id=“label-What+it+is%3F”&gt;What it is?&lt;span&gt;&lt;a href=“#label-What+it+is%3F”&gt;&amp;para;&lt;/a&gt; &lt;a href=“#documentation”&gt;&amp;uarr;&lt;/a&gt;&lt;/span&gt;&lt;/h3&gt;</p>

    <p>&lt;p&gt;&lt;strong&gt;doiref does two things:&lt;/strong&gt;&lt;/p&gt; &lt;ul&gt;&lt;li&gt; &lt;p&gt;Search for a paper. Uses the CrossRef Metadata Search API, which allows POST requests of free form text.&lt;/p&gt; &lt;/li&gt;&lt;li&gt; &lt;p&gt;Get a citation from a DOI. Uses CrossRef &lt;a href=“<a href=“citation”>labs.crossref.org/citation-formatting-service/“>citationlabs.crossref.org/citation-formatting-service/”>citation”>labs.crossref.org/citation-formatting-service/“>citation> formatting service&lt;/a&gt; to search for citation information.&lt;/p&gt; &lt;/li&gt;&lt;/ul&gt;</p>

    <p>&lt;p&gt;Each of the two above tasks are functions that you can use within Ruby, and are available from the command line/terminal so that you don&amp;#39;t have to spin up Ruby. This latter use case I think is really powerful. That is, during a typical writing workflow (in which you are using bibtex formatted references) one can want a citation for their paper, and instead of opening up a browser and using Google Scholar or Web of Science, etc., you can quickly search in your terminal by doing e.g., crsearch &amp;#39;keywords that will help find the paper, including author, year, etc.&amp;#39;. Which if matches will give you a DOI. Then you can do doiref DOI/string | pbcopy and you get the bibtex reference in your clipboard. Then just paste into your bibtex file or references manager.&lt;/p&gt;</p>

    <p>&lt;h3 id=“label-Dependencies”&gt;Dependencies&lt;span&gt;&lt;a href=“#label-Dependencies”&gt;&amp;para;&lt;/a&gt; &lt;a href=“#documentation”&gt;&amp;uarr;&lt;/a&gt;&lt;/span&gt;&lt;/h3&gt; &lt;ul&gt;&lt;li&gt; &lt;p&gt;HTTParty gem to make web calls to Crossref APIs&lt;/p&gt; &lt;/li&gt;&lt;li&gt; &lt;p&gt;bibtex-ruby gem to parse the bibtex&lt;/p&gt; &lt;/li&gt;&lt;li&gt; &lt;p&gt;json gem to convert to/from JSON&lt;/p&gt; &lt;/li&gt;&lt;/ul&gt;</p>
